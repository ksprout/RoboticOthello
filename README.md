# 🐱Robotic Othello🙊

オセロの対戦相手になってくれるPythonプログラムを作成してみました。

これから進化させていく予定です（遺伝的アルゴリズムを使用）

<br>

## 遊び方🐾
① プログラムを実行します
```
python main.py
```
② 自分の番が来たら石を置きます
```
● のターンです
```
この表示が出たら、あなたの番です。

例えば、0から数えて5行目かつ3列目の位置に置きたいときは
```
5 3
```
と入力します。

※ 途中で終了したいときは `q` と入力
